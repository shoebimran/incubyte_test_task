# Incubyte Assessment    

## Set up the project
#### Clone the Repo:   
`git clone https://github.com/ahmadadnan1602/incubyte_test_task.git`

#### change the directory:   

`cd incubyte_test_task`

#### Versions    
Ruby version: `3.1.0`   
Rails version: `7.1.5`

#### Install Dependencies:   
`bundle install`   
`rails db:create`     

`rails db:migrate`     

#### Run the Server:

`rails server`


#### Add RSpec to the Gemfile:

`gem 'rspec-rails'`   

run, 

`bundle install`

#### Initialize RSpec:    
`rails generate rspec:install`   

#### Run RSpec:
Now, you run using command:

`bundle exec rspec`   
or   
`rspec`
